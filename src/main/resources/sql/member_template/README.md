# 📂 개인 템플릿 (Member Template)

## 📖 개요
사용자가 생성한 템플릿을 저장하는 테이블입니다.  
템플릿은 공개(`public`), 비공개(`private`), 구독자 전용(`subonly`)으로 설정할 수 있으며,  
다른 템플릿 저장소(`template_repository`)와 연결될 수 있습니다.

---
