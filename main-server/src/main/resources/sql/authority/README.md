# 📂 구독 (Authority)

## 📖 개요
사용자 권한을 저장하는 테이블입니다.

---

## 🏗️ 테이블 구조

### 1️⃣ authority
> 사용자 역할 및 권한 정보를 저장하는 테이블

| 필드명      | 타입             | NULL 허용 | 기본값 | 설명 |
|------------|-----------------|----------|--------|------|
| `id`       | INT AUTO_INCREMENT | NO       | -      | 권한 ID |
| `name`     | VARCHAR(255)    | NO       | -      | 권한 이름 |
