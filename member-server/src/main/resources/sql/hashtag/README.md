# 📂 해시태그 (Hashtag)

## 📖 개요
게시글의 해시태그를 저장하는 테이블입니다.

---

## 🏗️ 테이블 구조

### 1️⃣ hashtag
> 해시태그 정보를 저장하는 테이블

| 필드명      | 타입             | NULL 허용 | 기본값 | 설명 |
|------------|-----------------|----------|--------|------|
| `id`       | INT AUTO_INCREMENT | NO       | -      | 해시태그 ID |
| `name`     | VARCHAR(255)    | NO       | -      | 해시태그 이름 |
